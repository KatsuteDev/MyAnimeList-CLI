⚠ This is an unofficial CLI for the MyAnimeList API

mal forum list

mal forum list <query> [flags]

Options

  • -b, --board <integer>

    The board ID

  • -s, --subboard <integer>

    The subboard ID

  • -t, --topic-username <string>

    The topic username

  • -u, --username <string>

    The author username

  • -l, --limit <integer>

    The maximum amount of results to return

  • -o, --offset <integer>

    The results offset

  • -c, --csv <fields>

    Output as CSV with a set of specified fields

  • -j, --json <fields>

    Output as JSON with a set of specified fields

  • -q, --jq <expression>

    Format JSON using jq syntax
